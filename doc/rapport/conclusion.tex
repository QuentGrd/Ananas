\section{Conclusion}
\label{sec:conclusion}
\subsection{Problèmes pencontrés}
\paragraph{}Lors de la réalisation de notre projet nous avons rencontré quelques problèmes comme:
\begin{itemize}
	\item La gestion des conflits sur Git
	\item Implémentation du design pattern Observer afin de synchroniser des éléments avec l'horloge
\end{itemize}

\subsection{Améliorations possibles}
\paragraph{}Voici une liste non exhaustive d'améliorations possibles du jeu:
\begin{itemize}
	\item Un Mode de jeu mélangeant personnages jouables et personnages autonomes
	\item Influence de l'émotion des personnages autour d'un autre personnage sur son émotion
	\item Gestion des embouteillages
	\item Optimisation de la consommation du processeur
	\item Modification des propriétés d'une infrastructure
	\item Modification des propriétés d'un personnage
	\item Création manuelle d'un personnage avec des propriétés personnalisées
	\item Développement d'une application Android
\end{itemize}

\subsection{Ressenti sur le projet}
\paragraph{}Ce projet nous a permis de développer nos compétences en programmation orientée objet, mais aussi sur la gestion d'un projet long. Nous voyons maintenant comment organiser le développement d'une application et comment gérer les différentes tâches. Ce projet nous a aussi permis de travailler sur des notions qui nous intéressent.
