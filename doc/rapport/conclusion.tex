\section{Conclusion}
\label{sec:conclusion}
\subsection{Problèmes Rencontrés}
\paragraph{}Lors de la réalisation de notre projet nous avons rencontré quelques problèmes comme:
\begin{itemize}
	\item La gestion des conflits sur Git
	\item Implementation du design pattern Observer afin de synchronisé des éléments avec l'horloge
\end{itemize}

\subsection{Améliorations Possible}
\paragraph{}Voici une liste non exhaustive d'améliorations possible du jeu:
\begin{itemize}
	\item Un Mode de jeu mélangeant personnages jouables et personnages autonomes
	\item Influence de l'émotion des personnages autour d'un autre personnage sur son émotion
	\item Gestion des embouteillages
	\item Optimisation de la consommation du processeur
	\item Modification des propriété d'une infrastructure
	\item Modification des propriété d'un personnages
	\item Création manuelle d'un personnage avec des propriétés personnalisés
\end{itemize}

\subsection{Ressenti sur le projet}
\paragraph{}Ce projet nous a permis de developper nos compétences en programmation orienté objet, mais aussi sur la gestion d'un projet long. Nous voyons maintenant comment organiser le developpement d'une application et comment gérer les différentes taches. Ce projet nous a aussi permis de travailler sur des notions qui nous intéressent.
